# Shigure Kira Blender Shader Preset

Tested on Blender 3.2. Modified upon [[Blender] 崩坏三格蕾修渲染展示](https://www.bilibili.com/video/BV1eA4y1o7rF/). Suitable for datamined `.fbx` models instead of MMD ones.
